> 名字取自阅尽好花千万树，愿君记取此一枝。
项目主要为个人学习项目，他的主要功能是分享网页链接，你也可以在查看内容的同时和其他人一起交流，另外还增加了和打印设备交互的功能，你可以在发布的内容里挖掘隐藏信息
